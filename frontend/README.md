# Fraud Detection Dashboard

Live dashboard for monitoring fraud detection in real-time.

## Features
- Real-time fraud probability analysis
- Transaction history tracking
- 8 fraud type monitoring
- Risk level visualization

## Usage
Open `index.html` in any modern browser.

The dashboard connects to the Fraud Detection API:
- API URL: https://fraud-detection-api-gzhw.onrender.com
- Endpoints: /stats, /predict, /fraud-types, /transactions

## Deployment
This is a static site that can be deployed on:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting
